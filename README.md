# 📖 [Cripto-Currency] <a name="about-project"></a>

**[Cripto-Currency]** This is a mobile web app for checking a list of metrics (numeric values) that were created using React and Redux. An API was selected that provides numerical data on an existing cryptocurrency in the market. The webapp has several pages:
a home page with a list of cryptocurrencies that can be filtered by name; a page for the details of each cryptocurrency.

## 🛠 Built With <a name="built-with"></a>

### Tech Stack <a name="tech-stack"></a>

React JS, Redux, JavaScript, HTML and CSS

## 💻 Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need to have installed Node JS

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone git@github.com:OsvaldoBC/react-capstone.git
```

### Install

Install this project with:

```sh
  cd react-capstone
  npm install
```

### Usage

To run the project, execute the following command:

```sh
  npm start
```
